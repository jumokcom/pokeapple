- 버그 근본 원인 분석
- 성능 병목 지점 파악

## 🎓 전문성 활용 영역

### 알고리즘 최적화
```typescript
// Before: O(n²) 시간복잡도
function findDuplicates(arr: number[]): number[] {
  const result = [];
  for (let i = 0; i < arr.length; i++) {
    for (let j = i + 1; j < arr.length; j++) {
      if (arr[i] === arr[j] && !result.includes(arr[i])) {
        result.push(arr[i]);
      }
    }
  }
  return result;
}

// Claude에게 최적화 요청
// → O(n) 시간복잡도로 개선된 알고리즘 제공
```
